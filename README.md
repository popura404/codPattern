# COD Pattern

[中文文档](docs/README.md) | [English README](docs/README.en.md) | [详细 Guide](docs/GUIDE.md) | [Q&A](docs/QANDA.md) | [更新日志](docs/CHANGES.md)

COD Pattern 是一个基于 **TaCZ + 内置 FPSM 兼容核心** 的 Forge 模组，提供 COD 风格的背包预设、局内改枪、`frontline / teamdeathmatch` 地图与房间流程，以及对应的服务端持久化和客户端 HUD 表现。

> 当前文档覆盖版本：`0.6.10b`
>
> 最近几个 Beta：
> - `0.6.7b`：动态复活点合并、开始投票前强制检查 `endtp`
> - `0.6.9b`：敌方血条改为世界空间渲染
> - `0.6.10b`：放宽传送安全校验，补充 `endtp` 失败告警、朝向预览，并将 `endtp set` 改为当前位置批量写入全部地图
